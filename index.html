Project 2

Daily Agneda

A to-do list app built with Python, Django, HTML, CSS, and JavaScript, using SQLite for data storage. Project 3

Blind Man Stick

A smart blind man’s stick designed to assist visually impaired individuals in navigation. It uses ultrasonic sensors to detect obstacles and provides real-time feedback through vibrations or audio alerts. The stick may also include GPS tracking for location assistance and an emergency alert system. This project enhances mobility and safety for visually impaired users.
